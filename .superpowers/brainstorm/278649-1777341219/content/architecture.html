Core Architectural Components

Dynamic Run Generator

Converts the AST into a stream of TextRuns. If the global cursor is within a block, it emits "Source" runs (syntax highlighted). Otherwise, it emits "Rich" runs (styled headers, hidden markers).

Global Offset Mapping

All interactions (clicks, keypresses) operate on a single document-wide UTF-8 byte offset. No more "block local" coordinates causing selection gaps.

Object Replacement (U+FFFC)

Inline Math and Typst blocks are treated as "objects" within the text flow. We reserve space in the layout and paint the SVG directly over the text canvas.
